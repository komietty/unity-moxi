# unity-moxi-ink

Lattice Boltzman Equation base ink simulation called Moxi implemented on Unity.

![img](Imgs/CaptureA.PNG)

## Compatibility

Tested on Unity 2019.3.0f1, windows10 (RTX 2080 max-q).
